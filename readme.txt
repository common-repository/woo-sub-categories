== What’s in it ==

Plugin will add a widget in your wordpress dashboard so you can use this to display subcategories on woocommerce categories page

== Installation ==

#### Thank you for your interest in Woocommerce Subcategory Widget.  

Here are two ways of installing WD Google Maps: via WordPress backend and via FTP. Let’s discuss each case individually:

**Installing via WordPress** 

Login to the WordPress Administrator Panel.  
Go to Plugins> Add New>Upload.   
Click Choose file button and select the zip folder of woocommerce-subcategory-widget plugin and press Install now button.   
Click Activate button.   

**Installing via FTP**

Login to your hosting space via an FTP software, e.g. FileZilla.   
Unzip the downloaded woocommerce-subcategory-widget plugin folder without making any changes to the folder.   
Upload the woocommerce-subcategory-widget into the following location wp-content>wp-plugins.   
Login to the WordPress Administrator Panel.   
Activate woocommerce-subcategory-widget by going to Plugins and pressing Activate button.
